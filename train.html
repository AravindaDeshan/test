Project Overview

The Train Booking System is a desktop-based application developed using Visual Studio, designed to simplify and streamline the process of booking train tickets. Built with a user-friendly graphical interface, the system provides an efficient platform for both passengers and administrators to manage train travel-related tasks.This project was developed as part of a software development curriculum to demonstrate practical knowledge of programming concepts, user interface design, and database integration.

Technology Stack

C# (Windows Forms) C# (Logic Layer) Microsoft SQL Server Microsoft Visual Studio

Key Features

  • User Registration & Login – Secure access for passengers and administrators.
  • Train Schedule Management – Add, update, and view train timings and routes.
  • Ticket Booking – Select source, destination, train, date, and class to book tickets.
  • Booking History – View past and upcoming bookings.
  • Seat Availability Check – Real-time seat status display.
  • Admin Panel – Manage users, trains, and schedules.

import nltk
from nltk.chat.util import Chat, reflections
import tensorflow as tf
from flask import Flask, request, jsonify

# Define conversation patterns
pairs = [
    [
        r"my name is (.*)",
        ["Hello %1, how can I help you today?",]
    ],
    [
        r"what is your name?",
        ["I'm the University Assistant bot. How can I help you?",]
    ],
    [
        r"how are you ?",
        ["I'm doing good! How about you?",]
    ],
    [
        r"(.*) (course|class) schedule",
        ["You can find course schedules at the university portal.",]
    ],
    [
        r"quit",
        ["Goodbye! Feel free to come back with more questions."]
    ]
]

# Initialize chatbot
chatbot = Chat(pairs, reflections)

# Create Flask app
app = Flask(__name__)

@app.route('/chat', methods=['POST'])
def chat():
    user_input = request.json.get('message')
    response = chatbot.respond(user_input)
    return jsonify({'response': response})

if __name__ == '__main__':
    app.run(debug=True)
